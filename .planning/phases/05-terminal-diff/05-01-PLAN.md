## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| WebView → Tauri invoke | User keyboard input forwarded to PTY; must scope writes to known terminal id |
| Rust PTY → Channel → xterm | Raw shell output including ANSI; no filtering per D-37 — trust local shell |
| Agent message → diffParser | Untrusted diff text parsed client-side; read-only render, no file writes (D-23) |

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05-01 | Tampering | write_to_terminal | mitigate | Validate terminal id exists in TerminalManager before write |
| T-05-02 | DoS | Channel streaming | accept | scrollback 10000 cap; local dev app, low risk |
| T-05-03 | Spoofing | diffParser | accept | Display-only; no apply/reject actions in Phase 5 |
| T-05-04 | Tampering | npm PTY/xterm/diff packages | mitigate | Use pinned versions in package.json/Cargo.toml; run cargo build + tsc before ship |
| T-05-SC | Tampering | portable-pty, @xterm/xterm, @git-diff-view/react installs | mitigate | Versions pinned in manifest; verify crates.io/npmjs.com names match before first install |
</threat_model>
