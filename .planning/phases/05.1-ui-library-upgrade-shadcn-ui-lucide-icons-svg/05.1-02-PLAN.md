<action>
在 apps/desktop 目录运行 shadcn init：
```bash
cd apps/desktop && npx shadcn@latest init -d --style new-york --base-color neutral
```
参数说明：
- `-d`: 使用默认值（CSS variables: yes）
- `--style new-york`: new-york 风格
- `--base-color neutral`: 中性灰色系

CLI 会自动：
1. 创建 components.json
2. 更新 src/index.css 为 shadcn CSS 变量体系
3. 创建 src/lib/utils.ts（cn 函数）
4. 安装依赖：class-variance-authority, clsx, tailwind-merge, lucide-react, tw-animate-css
</action>

<action>
验证并手动调整 index.css：
1. 确认 `@import "tailwindcss"` 和 `@import "tw-animate-css"` 存在
2. 确认暗色模式变体为 `@custom-variant dark (&:is(.dark *))`
3. 确认 `@theme inline { ... }` 包含所有 CSS 变量映射
4. 确认 `@layer base { * { @apply border-border; } body { @apply bg-background text-foreground; } }`
5. 移除旧的 paladin 自定义色阶（已被 shadcn 覆盖）
6. 移除旧的 `@variant dark (&:where(.dark, .dark *))`（已被 shadcn 替代）
7. 移除 `@keyframes slide-up` 和 `.animate-slide-up`（将在 Wave 3 中由 Sonner 动画替代）
8. 移除旧的 `body { @apply bg-white dark:bg-gray-900 text-gray-900 dark:text-gray-100; }`（已被 `bg-background text-foreground` 替代）

注意：如果 shadcn init 已经正确处理了上述内容，则只需验证。
</action>
