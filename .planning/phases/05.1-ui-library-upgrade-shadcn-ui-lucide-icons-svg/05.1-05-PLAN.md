<action>
重构 RightDrawer.tsx 为 Sheet 实现：
1. 导入 Sheet 组件：
```tsx
import {
  Sheet, SheetContent, SheetHeader, SheetTitle,
} from '@/components/ui/sheet';
```

2. 用 Sheet 替换外层 div：
```tsx
<Sheet open={isOpen} onOpenChange={(open) => {
  if (!open) useTerminalStore.getState().closePanel();
}} modal={false}>
  <SheetContent side="right" className="w-[400px] sm:max-w-[600px] p-0 flex flex-col">
    <SheetHeader className="px-3 py-2 border-b border-gray-200 dark:border-gray-700">
      <SheetTitle className="text-xs font-medium">
        {activePanel === 'terminal' ? '终端' : '代码变更'}
      </SheetTitle>
    </SheetHeader>
    <div className="flex-1 flex flex-col min-h-0">
      {activePanel === 'terminal' && <TerminalTabBar ... />}
      <TerminalPanel channel={channel} />
    </div>
  </SheetContent>
</Sheet>
```

3. 关键适配：
- `modal={false}` — 不让 Sheet 拦截焦点（终端需要键盘输入）
- 移除 ResizeHandle（Sheet 不支持动态拖拽宽度）
- 保留 Tauri Channel 生命周期、Tab 管理逻辑
- 保留 TerminalPanel 和 TerminalTabBar 内部逻辑

4. 移除旧的 width transition 动画（Sheet 自带滑入/滑出动画）
5. 移除 `flex-shrink-0`, `border-l`, `overflow-hidden`, `transition` 等 class
</action>
