## Top 3 Priority Fixes

1. **大量 `bg-gray-*`/`text-gray-*` 未迁移到 shadcn 色系 token** — 视觉与 shadcn 主题脱节，暗色模式下颜色不一致 — 将 App.tsx、ConversationList、Titlebar 中的 `bg-white` → `bg-background`、`text-gray-500` → `text-muted-foreground`、`border-gray-200` → `border-border` 等逐一替换
2. **删除操作使用原生 `window.confirm`** — 破坏视觉一致性，无品牌化体验 — 替换为 sonner toast 确认或 shadcn Dialog/AlertDialog
3. **ResizeHandle.tsx 成为死代码** — 编译无报错但已无引用（Sheet 不支持拖拽宽度） — 删除文件以保持代码库整洁

### Pillar 4: Typography (4/4)

**使用字号：** `text-xs` (6), `text-sm` (12), `text-base` (1), `text-2xl` (1), `text-3xl` (1)

**使用字重：** `font-medium` (7), `font-semibold` (1), `font-bold` (4)

**结论：** 字号和字重控制在合理范围内（≤5 种字号，≤3 种字重）。字体为 Geist Variable，由 shadcn 统一管理，无需调整。

### Pillar 5: Spacing (3/4)

**最常用间距类：** `px-*` (23), `p-*` (19), `gap-*` (18), `py-*` (15)，以 `px-3`, `py-2`, `gap-2`, `p-1.5` 为主 — 模式一致。

**自定义值：** 
- `w-[400px] sm:max-w-[600px]` — RightDrawer Sheet 宽度（[RightDrawer.tsx:83](file:///Users/worsmer/Workspace/paladin/apps/desktop/src/components/layout/RightDrawer.tsx#L83)），合理
- shadcn ui 组件内部 `[inherit]`/`[3px]` 等 — 库内部值，非本代码问题

**问题：**
- 无重大问题，间距模式一致
