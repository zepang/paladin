# Phase 05.1: UI-SPEC — UI Library Upgrade

**Generated:** 2026-06-17
**Style:** new-york (shadcn/ui default)
**Icon Library:** lucide-react

## Visual Contract

This phase replaces custom components with shadcn/ui defaults. The visual contract is **"shadcn/ui new-york style with neutral base color"** — no custom design work.

### Component Replacement Map

| Original | Replacement | Visual Note |
|----------|------------|-------------|
| ChatToggle (手写 button + SVG) | `Button variant="ghost" size="icon"` + lucide `MessageSquare` | 保持原尺寸和交互位置 |
| ThemeToggle (手写 button + SVG) | `Button variant="ghost" size="icon"` + lucide `Sun`/`Moon`/`Monitor` | 保持三态循环切换 |
| Titlebar window buttons | `Button variant="ghost" size="icon"` + lucide `Minus`/`Maximize2`/`X` | 保持 Titlebar 右侧布局 |
| StatusBar terminal toggle | `Button variant="ghost" size="sm"` | 保持绿色圆点指示器 |
| ErrorToast | `Sonner` toast | 底部居中，保持错误/警告/信息三态 + 重试按钮 |
| ConversationList 滚动 | `ScrollArea` | 仅替换外层滚动容器，内部保留 |
| RightDrawer | `Sheet side="right"` | **高风险** — 如果焦点/布局不兼容则回退 |

### 不变组件

以下组件视觉和行为完全不变：MessageBubble, MessageList, WelcomePage, ChatView, TerminalPanel, DiffMessageCard, TerminalTabBar

### 色系

- 废弃现有 paladin-50~900 蓝紫色
- 使用 shadcn neutral OKLCH 色系
- `--primary` 默认使用 neutral 深色（黑色系）

### 暗色模式

- `.dark` class 策略不变
- Zustand theme store 继续管理
- shadcn `@custom-variant dark (&:is(.dark *))` 替代当前 `&:where(.dark, .dark *)`

### Icon 尺寸

- 所有 lucide icon 使用 `size-4`（16px），与当前 SVG 尺寸一致
