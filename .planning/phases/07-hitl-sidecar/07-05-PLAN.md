  <action>
    创建 ApprovalBridge.tsx — 审批事件桥接组件：

    1. TypeScript 类型定义（文件顶部）：
       ```typescript
       interface ApprovalRequest {
         request_id: string;
         tool_name: string;
         args: Record<string, unknown>;
         reason: string;
       }
       interface ApprovalState {
         pendingApprovals: ApprovalRequest[];
         current: ApprovalRequest | null;
         isConnected: boolean;
       }
       ```

    2. `useApprovalBridge()` hook：
       - useEffect 创建 EventSource('http://localhost:9876/approval/stream')
       - onmessage: 解析 JSON → 追加到 pendingApprovals 队列（FIFO）
       - onerror: 设置 isConnected=false，5s 后自动重连
       - 组件卸载时 eventSource.close()

    3. 队列管理：
       - `current = pendingApprovals[0]`（队首为当前审批）
       - `approve(requestId)` → fetch POST /approval/{requestId} body {decision: true} → 出队
       - `deny(requestId)` → fetch POST /approval/{requestId} body {decision: false} → 出队
       - 回调后从 pendingApprovals.shift() 移除队首，next pending 自动成为 current

    4. 导出：
       - `useApprovalBridge()` — 组件 hook（React.FC 嵌入 App.tsx）
       - ApprovalBridge 返回 null（无自身 UI，仅管理状态）
       - 状态通过 React Context 或简单 props 向下传递

    5. 错误处理：POST 失败时 toast.error('审批提交失败')，不阻塞队列。
  </action>
