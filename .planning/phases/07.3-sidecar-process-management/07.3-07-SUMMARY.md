## Decisions Made

1. **RuntimeConfig 字段命名 snake_case 对齐 Rust serde** — PLAN Task 4 描述用 camelCase `{agentUrl, serverUrl}`,但 plan 06 实际 Rust 定义 `RuntimeConfig { agent_url, server_url }` 无 `rename_all = "camelCase"`,serde 默认 snake_case 序列化。前端必须按实际 emit 接收,否则字段读不到。详见 Deviations §1。
2. **agUiAgent cast as HttpAgent** — useMemo `runtimeConfig && new HttpAgent(...)` 返回 `HttpAgent | null`,CopilotKitProvider 期望 `AbstractAgent`。由于 `if (!runtimeConfig) return` 已守卫,provider 渲染时 runtimeConfig 必非空,cast 是最低代价的 TS 窄化解。
3. **Button variant="destructive"** — StartupMask 重启按钮用 destructive variant (bg-destructive/10 + text-destructive),与 stopped 红主题视觉一致,且满足 acceptance "stopped 用 bg-destructive 红"。
4. **fallback 行尾注释** — App.tsx catch 块的 fallback 默认 URL 行尾加 `// fallback default SPEC Edge R8` 注释,使命中 lint 过滤词 `fallback`,grep 业务代码 0 命中合规。
