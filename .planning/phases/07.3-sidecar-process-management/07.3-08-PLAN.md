<objective>
把 StatusBar 从「单终端灯」升级为「终端灯 + Agent 灯 + Go 灯」三灯一排，并为 Agent/Go 灯加 Popover 控制菜单（restart/stop/查看日志 + last_error + stderr_tail）。本 plan 是 D-09/D-11/D-12 的 UI 落地点，复用 plan 04 的 Popover 组件与 plan 07 的 process store。

D-09 关键：用户在讨论中纠正了「ChatToolbar 升级」的初版建议——ChatToolbar 在 Agent offline 时随主界面整个不渲染，状态灯放那里在「最需要看状态时」反而看不到。StatusBar 始终渲染（plan 07 D-10 已确保），所以灯放 StatusBar。

Purpose: 让用户在任意状态（启动/异常/停止）都能看到两个子进程的实时状态 + 一键控制。
Output: StatusBar 三灯可见；点 Agent/Go 灯弹 Popover；restart/stop 按钮工作；查看日志跳 logs 视图。
</objective>
