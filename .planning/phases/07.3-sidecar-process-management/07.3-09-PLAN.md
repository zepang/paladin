## Artifacts this phase produces

**components/panel/LogsPanel.tsx：**
- `export function LogsPanel()`
- 内部 state：`activeTab: 'agent' | 'server'`（默认 'agent'）；`buffers: { agent: string[], server: string[] }`（用 useRef 持有，避免 re-render 风暴；或 useReducer 批量更新）；`paused: boolean`；`autoScrollRef` 指向滚动容器。
- listen('process-log') — payload LogChunk { process: 'agent'|'server', stream: 'stdout'|'stderr', line: string, ts: number }；按 process append 到对应 buffer（stderr 行前加红色标记或 ⚠️ 前缀以视觉区分，但不做级别折叠）。
- autoScroll：滚动容器 onScroll 检测 near-bottom（threshold 3 行），near 时 autoScroll=true 否则 false；新行 append 时若 autoScroll 则 scrollTop = scrollHeight。
- 暂停：paused=true 时仍 listen 但不 append（或 append 到 buffer 但不滚动）；Play 恢复。
- 清空：清当前 activeTab 的 buffer。
- UI：顶部 tab 切换（paladin-agent / paladin-server Button）+ Pause/Play + Trash2 图标按钮；下方滚动区 `<pre>` 或 `<div>` 白空间 pre-wrap 显示 buffer.join('\n')。
