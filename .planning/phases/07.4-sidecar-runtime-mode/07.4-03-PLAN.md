## Artifacts This Phase Produces

- `ProcessOwner = 'supervisor' | 'external' | 'none'`
- `ProcessHealth = 'healthy' | 'degraded' | 'failed' | 'unknown'`
- `ProcessState` includes `'conflict'`
- `ProcessInfo.owner`
- `ProcessInfo.health`
- `ProcessLight` ownership label map, health/state diagnostic labels, disabled external stop/restart controls, `redetect_${name}` action
- `StartupMask` support for `conflict` and failure classes: executable missing, cwd missing, port conflict, health failure, startup-grace exit, readiness degraded
- Config diagnostic rendering path if Rust exposes global config failure through status or runtime config diagnostics
- Store/component tests for idempotent tuple merge, external action UX, conflict diagnostics, and Go-not-blocking rule
