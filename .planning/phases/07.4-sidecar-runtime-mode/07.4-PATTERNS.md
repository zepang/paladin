- Rust remains the runtime truth source. Extend `ProcessState` in `supervisor.rs` lines 99-113 with `Conflict`, and add `ProcessOwner` / `ProcessHealth` serde-lowercase enums next to it.

- Current `spawn_child_to_slot()` performs bind-first at `supervisor.rs` lines 496-500. 07.4 must use health-first in dev mode:

  - liveness failed + port occupied -> conflict/none/failed, emit status, no spawn.
- Reuse URL construction and expected-status logic from `probe_loop()` lines 748-764 and 789-812.
- Use port helpers only after health failure; `port_probe.rs` lines 17-40 remain the port fact layer.
