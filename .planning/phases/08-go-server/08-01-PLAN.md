## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| environment -> config | Untrusted env vars drive ports, DSNs, and secrets before any handler runs. |
| config -> server bootstrap | Short/missing secrets would weaken HS256 or fail DB/Redis connects silently. |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-08-01-01 | Spoofing | JWT secret strength | high | mitigate | config.Load rejects PALADIN_JWT_SECRET shorter than 32 bytes (RESEARCH Pitfall #4). |
| T-08-01-02 | Information Disclosure | .env.example | medium | mitigate | Placeholder-only values; .gitignore excludes the real .env (D-14). |
| T-08-01-SC | Tampering | package installs | low | accept | Module proxy + sum.golang.org checksums (go mod verify) lock dependency integrity. |
</threat_model>
