- Updated `apps/server/internal/http/ws/handler.go` — `Handle` passes the roles from the JWT context (set by Phase 8 Auth middleware) into `hub.Register(wc, roles)`; preserves the ping loop, read loop, and JWT rejection. Unknown inbound envelope types from clients are logged and ignored (no panic).
- `apps/server/internal/ws/hub_test.go` — full test suite (see feature behavior).
