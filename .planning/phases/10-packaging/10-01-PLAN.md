<threat_model>
- ASVS level: 1
- Block on severity: high
- Threat: packaged app silently falls back to dev commands or PATH tools. Mitigation: D-04/D-06 force explicit sidecar production and early failure before bundling.
- Threat: shell injection or environment-dependent build behavior sneaks into packaging. Mitigation: keep package-manager arrays and `spawn(..., shell: false)` patterns; never switch to shell strings.
- Threat: release pipeline claims legitimacy without both sidecars present. Mitigation: artifact existence gates block the build.
- Threat: Agent packaging accidentally bakes or prints secrets during build or docs. Mitigation: do not embed real environment values in scripts, comments, or test fixtures.
</threat_model>
