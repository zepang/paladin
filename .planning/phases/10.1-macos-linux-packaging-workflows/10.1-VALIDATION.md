## Sampling Rate

- **After every task commit:** Run `node --test apps/desktop/scripts/build-manifest.test.mjs apps/desktop/scripts/release.test.mjs`
- **After every plan wave:** Run `pnpm test`
- **Before `$gsd-verify-work`:** Full suite must be green, static workflow path audit must pass, docs wording audit must pass, and manual CI evidence must record exact installer counts.
- **Max feedback latency:** 10 seconds for quick script tests; full suite latency accepted at wave boundaries.
