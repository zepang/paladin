## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Provider form -> store | Raw key input exists transiently in React component state. |
| Store -> Tauri wrappers | Provider mutations cross to Rust command layer. |
| Masked readback -> rendered UI | Provider metadata becomes user-visible text and test snapshots. |

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |

| T-11-06-02 | Tampering | aiProvider store | medium | mitigate | All mutations use typed Tauri wrappers; save/test are separate actions with explicit active selection. |
| T-11-06-03 | Spoofing | readiness labels | medium | mitigate | Render AI readiness vocabulary only, not process health labels. |
